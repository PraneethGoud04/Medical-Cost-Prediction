# Healthcare Cost Prediction Using Machine Learning

## Overview
This project predicts annual healthcare expenditure using Machine Learning techniques based on healthcare risk indicators, hospitalization history, insurance claims, and chronic disease information.

## Features
- Machine Learning Regression Models
- Random Forest Regressor
- Streamlit Interactive Dashboard
- Healthcare Analytics
- Real-Time Cost Prediction

## Technologies Used
- Python
- Pandas
- Scikit-learn
- Streamlit
- Joblib

## Model Performance
- High R² Score
- Low MAE
- Cross Validation & Hyperparameter Tuning

## Deployment
Streamlit Dashboard for real-time healthcare cost prediction.
